# Pingay — Network Diagnostic Tool

Uma ferramenta premium de diagnóstico de rede para profissionais de TI, desenvolvida em Electron.

## Funcionalidades
- **Monitor de Ping:** Acompanhamento em tempo real de múltiplos hosts com notificações de queda.
- **Traceroute:** Visualização detalhada de saltos até o destino.
- **DNS Lookup:** Consulta de registros A, AAAA, MX, NS e PTR.
- **Port Scanner:** Varredura rápida de portas comuns e personalizadas.
- **Histórico:** Log persistente de eventos de conectividade.
